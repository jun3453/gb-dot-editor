# ゲームボーイ向けドット絵作成ツール (GBDotEditor) 開発計画

ゲームボーイ (Game Boy) のタイル/スプライト画像を作成するための、ウェブベースのモダンなドット絵エディタを開発します。
古いツールである Gameboy Tile Designer (GBTD) の機能を踏襲しつつ、UI/UXを大幅に刷新し、現代的で直感的な操作、およびアニメーション作成が可能なWebツールを構築します。

## 1. 目的と要件
- **サイズ対応**: 8x8ピクセルを基準とし、最大256x256ピクセルまで（8の倍数）設定可能にします。
- **ゲームボーイのカラー制限**: GBの実機制限に合わせた4つの階調（カラーパレット：ホワイト、ライトグレー、ダークグレー、ブラック）で描画を行います。
- **アニメーション機能（新規追加）**:
  - パラパラ漫画形式で、複数の「フレーム」を作成・管理できます。
  - フレームの追加、削除、複製（現在のドット絵をコピーして新規作成）、順序の入れ替えが可能。
  - アニメーションの再生/一時停止、再生速度（FPS）の変更、ループ再生機能。
  - オニオンスキン機能（前後のフレームをうっすらと半透明で重ねて表示する機能）により、滑らかなアニメーション作成をサポート。
- **GBDK 2020対応**: GBDK2020（C言語）でそのまま使用できる形式（2bpp、8x8タイルごとに16バイトの配列データ）でのエクスポートおよびインポート。
  - 複数フレームがある場合は、フレーム0から順番に全てのフレームのタイルデータを連続した配列としてエクスポートします。
- **プロジェクト保存**: 編集中のデータ（全フレームの情報を含む）を独自のJSON形式でインポート・エクスポートし、作業を再開できるようにします。
- **UI/UXの改善**:
  - レスポンシブで現代的なデザイン（ダークテーマ基調、洗練された配色、流麗なアニメーション）。
  - ペン、消しゴム、塗りつぶし、スポイト、反転、回転、シフトなどのドット絵編集機能の充実。
  - タイル単位のグリッドとピクセル単位のグリッドの視覚的な差別化。
  - アニメーションおよび静止画のリアルタイムプレビュー。

## 2. 開発手順
1. **設計と仕様決定**
   - 画面レイアウト、アニメーションタイムラインUI、データ構造の決定。
   - `implementation_plan.md` の作成とユーザー承認。
2. **基盤の実装**
   - `index.html`, `style.css`, `app.js` のスケルトン作成。
   - GBカラーパレット、キャンバス状態管理の実装。
3. **エディタコア機能の実装**
   - Canvas APIを用いたピクセル描画（ペン、消しゴム、バケツ塗りつぶし、スポイト）。
   - キャンバスサイズ変更機能（8x8 〜 256x256）。
   - 各種変形ツール（上下左右反転、回転、上下左右シフト）。
4. **アニメーション機能の実装**
   - 複数フレーム管理（配列によるフレーム管理、追加・コピー・削除・移動）。
   - タイムラインUIの実装（フレーム一覧のサムネイル表示、選択）。
   - アニメーション再生エンジン（`requestAnimationFrame` または `setInterval` によるプレビュー描画）。
   - オニオンスキン表示ロジックの実装。
5. **インポート・エクスポート機能の実装**
   - JSONファイルによる全フレーム情報のインポート・エクスポート。
   - GBDK2020互換のC言語コード形式（`unsigned char` 配列）でのマルチフレームデータのエクスポート。
   - GBDK2020互換のC言語コードのインポート（複数フレームへの自動分割復元）。
   - PNG画像（スプライトシート、または個別フレーム）としてのエクスポート。
6. **UI/UXのブラッシュアップ**
   - CSSによる洗練されたデザイン（ガラスモフィズム、微細なアニメーション）。
   - 操作キーのショートカット対応。
7. **動作確認と検証**
   - 各種サイズでの描画、アニメーション動作、C言語コードの出力検証。
   - walkthrough.md の作成。

## 3. 作成・修正対象ファイル

- `style.css` (デザインスタイル)
- `app.js` (エディタロジック、アニメーション制御、インポート・エクスポート、描画処理)
- `Plan.md` (本ドキュメント)
- `AICHANGELOG.md` (変更履歴記録用)

## 4. 進捗と追加機能
- **ピクセルグリッド色のカスタマイズ機能 (2026-06-19 実装)**:
  - ピクセルグリッドの色をユーザーが自由に変更できるようにカラーピッカーUIを追加。
  - デフォルトの色味を薄いグレー (`#888888`) に設定。
  - 選択したグリッド色は `localStorage` に保存され、リロード後も永続化されるように実装。
- **ショートカットキーの強化およびコピペ・選択範囲ツールの実装 (2026-06-19 計画中)**:
  - 1~4キーでのパレットカラー選択、[ ]キーでのフレーム移動ショートカット。
  - 選択ツールによる矩形選択、コピペ（ドラッグ移動可能な貼り付けプレビュー）、選択解除機能の実装。
- **README.md の作成 (2026-06-19 計画中)**:
  - GitHub連携に伴い、プロジェクト概要、特徴、使用方法、ファイル構造などを記載した日本語の README.md を作成。
